# Calculadora

Uma calculadora simples e responsiva desenvolvida com HTML, CSS e JavaScript. Permite realizar operações básicas de matemática como adição, subtração, multiplicação, divisão e porcentagem.

## Funcionalidades

- **Operações básicas**: Adição (+), subtração (-), multiplicação (×), divisão (÷) e porcentagem (%).
- **Interface intuitiva**: Botões grandes e fáceis de usar.
- **Responsiva**: Funciona bem em dispositivos móveis e desktop.
- **Limpar e apagar**: Botão C para limpar tudo e ⌫ para apagar o último dígito.
- **Tratamento de erros**: Exibe "Erro" em caso de expressão inválida.

## Como Usar

1. Baixe ou clone o repositório.
2. Abra o arquivo `index.html` em qualquer navegador web moderno (Chrome, Firefox, Safari, etc.).
3. Clique nos botões numéricos para inserir números.
4. Clique nos botões de operadores para selecionar a operação.
5. Pressione o botão "=" para calcular o resultado.
6. Use "C" para limpar o visor ou "⌫" para apagar o último caractere.

## Tecnologias Utilizadas

- **HTML**: Estrutura da página e elementos da interface.
- **CSS**: Estilização visual, incluindo gradientes, sombras e responsividade.
- **JavaScript**: Lógica para manipulação do visor, entrada de dados e cálculo de expressões usando `eval()`.

## Estrutura do Projeto

- `index.html`: Arquivo principal contendo HTML, CSS e JavaScript.

Compatível com navegadores modernos que suportam HTML5, CSS3 e JavaScript ES6+.
